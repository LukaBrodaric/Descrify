# Descrify

**Descrify** is an AI-powered web application that generates SEO-optimized property descriptions for real estate listings and vacation rentals.

The tool helps real estate agents, property owners, and rental hosts quickly create high-quality listing descriptions ready for publication on property portals.

### Descrify live app - http://descrify.xyz/

---

## Features

- Generate **SEO-optimized property descriptions**
- Create **short and long listing descriptions**
- Generate **SEO titles and suggested keywords**
- Optional **image analysis for detecting property features**
- Structured outputs ready for **real estate portals**
- Simple **rate limiting system**
- Clean and modern **SaaS-style UI**

---

## Example Output

The AI generates structured content such as:

- SEO title
- Short description
- Long description
- Suggested keywords
- Image-detected features

All content is generated based on the property details and images provided by the user.
